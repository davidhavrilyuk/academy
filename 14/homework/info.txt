Домашнее задание №14
Сделать рабочую форму регистрации для интернет магазина. Для верстки можно использовать элементы CSS фреймворка Bootstrap (http://getbootstrap.com/components/)
Форма соостоит из 4 шагов, после правильного заполнения предыдущего, пользователь переходит на следующий шаг. Перейти на следующий шаг, можно только в том случаи, если все данные заполены верно (согласно валидации). Валидацию делать, после заполнения каждого элемента, в случаи ошибки, сразу выдавать пользователю уведомление и подсвечивать рамку элемента красным цветом.
На первых 3 шагах пользователь заполняет данные и на 4 шаге все эти данные выводятся для проверки, после чего нужно отметить чекбокс, о том что согласен с правилами и станет доступна кнопка "Финиш", которой можно закончить регистрацию.
Все шаги формы должны быть на одной странице, без перезагрузки. Все данные сохранять между шагами в LocalStorage, после нажатия на "Финиш" форма должна создать строку с данными(key=value) и записать в cookie.
Валидация:
Login - мин 8 символов
Password - мин 8 символов, обязательно присутствие большой буквы и цифры
Phone - в междунаробном формате:  + код номер
Card number - 16 цифр
Остальные поля без валидации
